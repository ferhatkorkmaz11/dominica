CS315 Project Part-1 ~ A Programming Language for Sets and its Lexical Analyzer
Dominica Programming Language

How to compile and run our example programs:
1-) Extract the dominica.l and example programs into a folder.
2-) cd into that folder.
3-) type "lex dominica.l"
4-) Then, type "g++ -o <output_filename> lex.yy.c"
5-) After, that you can test our example programs with typing "./<output_filename> < example_programX" where  1 <= x and x <= 5
